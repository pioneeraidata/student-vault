# 🏫 Classes MOC

MOC means "Map of Content." It is a page whose whole job is to link to other pages. This one is the map of every class you take. Start here when you want anything school related.

Rename the example classes below to your real schedule. Each class has its own page (its own mini MOC) that collects every lecture note, assignment, and study guide for that subject.

## How a class flows

1. In class, make a **Class Note** for the day (use the Class Note template). One note per lecture or discussion.
2. Homework and tests become notes in [Assignments](../20%20-%20Assignments/Assignments%20Dashboard.md), tagged with the class.
3. Before a test, build a [Study Guide](../30%20-%20Study/Study%20Hub.md) that pulls the class notes together.

Every class note links back to its subject MOC, so each subject page slowly becomes a complete record of the course.
