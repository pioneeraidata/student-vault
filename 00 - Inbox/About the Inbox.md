# 📥 About the Inbox

This is the junk drawer, and that is a good thing.

When a thought hits and you do not have time to file it, drop it here and keep moving. A homework reminder, a random idea, a link, a quote, a question you want to ask a teacher. Speed beats tidiness. A messy note that exists beats a perfect note you never wrote.

## How to use it

1. Capture anything, any format. One line is fine.
2. Once a day or once a week, look at what piled up and move each item to its real home:
   - A task or due date, go to [Assignments](../20%20-%20Assignments/Assignments%20Dashboard.md)
   - A class thought, go to that [class page](../10%20-%20Classes/Classes%20MOC.md)
   - An idea or goal, go to [Life](../50%20-%20Life/Life%20Hub.md)
3. Delete what does not matter anymore. Deleting is winning.

The [Weekly Review](../90%20-%20System/Templates/Weekly%20Review.md) is when the inbox gets emptied. If you only clean it once a week, that is still fine.
