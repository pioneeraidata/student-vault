# The Student Vault

A complete, ready-to-open Obsidian vault built for a high school student. Plain text, owned forever, no accounts, no subscription, works offline. Open this folder in Obsidian with "Open folder as vault" and it just works.

## What makes it good for a teenager

- **One habit, not a system to maintain.** The daily note is the front door. Everything else is optional structure that pays off when they need it.
- **Built around the real jobs of high school:** classes, assignments and due dates, studying for tests, reading, activities, and college prep.
- **Teaches thinking, not busywork.** It is wired around a simple loop: Capture (voice to text), Keep (this vault), Understand (an AI that answers only from your own sources, with citations).
- **Vanilla Obsidian only.** No community plugins required, so it opens on any machine and never breaks. Two core plugins do the work: Daily Notes and Templates, both pre-configured.
- **Worked examples everywhere.** Every folder ships with a filled-in example so a student sees the pattern instead of a blank page.

## How to hand it off

1. Copy or move this `StudentVault` folder anywhere (Desktop, Documents, a synced drive).
2. In Obsidian, choose "Open folder as vault" and point it at this folder.
3. Have the student read `Start Here.md`, then `90 - System/Guides/How to Use This Vault.md`.
4. Rename the example classes in `10 - Classes/` to their real schedule. Done.

## How to personalize it fast

- Replace the five example subjects (English, Math, Science, History, Spanish) with the student's actual classes.
- Delete any example notes marked "Example -" once the pattern is clear.
- Set the daily note hotkey and the template hotkey (see `90 - System/Guides/Setup - Turn On These Plugins.md`).

## Structure

```
Start Here.md            Front door / dashboard
00 - Inbox/              Fast capture, sort later
01 - Daily/              Daily notes (pre-wired as the Daily Notes folder)
10 - Classes/            One page per class + lecture notes
20 - Assignments/        Every due date in one place
30 - Study/              Study guides + self-quizzes
40 - Reading/            Book and article notes
50 - Life/               Goals, clubs, friends, ideas, and the weekly time plan
60 - College & Future/   College list, essays, activities, deadlines
70 - Athletics/          Seasons, games, goals, and getting better
80 - Music/              Instruments, lessons, pieces, performances
90 - System/             Templates, Guides, MOCs, and vault config
```

Time management is treated as a first-class layer, not a folder. `My Commitments`
(fixed weekly schedule), the `Weekly Planner` template, and the `Managing Your Time`
guide help a student see collisions (in-season plus a big test plus a recital plus a
social plan) before they happen, and keep friends from getting squeezed out.

House rule inherited from the design: no em dashes anywhere in the notes.
