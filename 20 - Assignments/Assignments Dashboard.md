# ✅ Assignments Dashboard

Every homework, project, quiz, and test lives here as its own note, so nothing falls through the cracks. This page is the one place you check to answer "what is due and when."

## How it works

1. New assignment, make a note with the **Assignment template** (or duplicate an example below).
2. Fill the top: class, due date, and status (`todo`, `doing`, or `done`).
3. When you touch it, update the status. When it is graded, log the grade so you can spot patterns later.

Keep the table below updated by hand. It takes ten seconds and it is the whole reason you never blank on a deadline.

## The one rule that saves you

The moment a teacher says a due date, write it here before you leave the room. Not after school. Not tonight. Now. A due date you did not capture is a due date you will miss.

> Power move: if you turn on the community plugin Dataview later, this table can build itself from the status fields in each note. It is optional. The by-hand table works fine and never breaks.
