# 🔗 Linking & MOCs 101

The one thing that turns a folder of notes into a second brain. Two minutes to learn.

## Links

Type two square brackets and start writing a note's name: `[[English MOC]]`. Obsidian suggests matches as you type. Pick one and you have made a link.

Now the magic part. Open that English page and scroll to the bottom. There is a **Backlinks** panel showing every note that links to it. You did not build that list. It built itself, the moment you made the links. Link your daily notes and assignments to a class, and that class page slowly becomes a complete record of everything you did in it, with zero extra work.

Rule of thumb: **when you mention something that has (or could have) its own note, link it.** A class, an assignment, a book, a person, an idea. Do not overthink it. Link generously.

## MOCs (Maps of Content)

A MOC is just a note whose job is to link to other notes. A table of contents you make yourself. This vault has a few:

- [[Start Here]] is the top level map
- [[Classes MOC]] maps every class
- Each subject page (like the English page) is a mini MOC for that class

When a folder gets busy, make a MOC for it. It is the map you check first. You do not need many. Start with the ones here and add one only when you feel lost in a pile of notes.

## The graph

Click the graph icon in the left sidebar. Every note is a dot, every link is a line. Early on it looks sparse. After a semester of linking, it looks like a brain, because it is a picture of yours. It is also a fast way to spot a note that is floating alone with no links, which usually means it is lost.
