# 🧭 How to Use This Vault

Read this once. It is short on purpose.

## The whole system in three lines

1. Open today's **daily note** and write in it. Every school day.
2. When something deserves a better home (a class, an assignment, a book), make a note for it and **link it** with `[[double brackets]]`.
3. Once a week, do a **Weekly Review**. Clean up, look ahead, done.

That is it. Everything else in this vault is a nicer home for stuff, or a guide like this one. You could ignore all of it and just use the daily note and still be ahead of almost everyone.

## The idea behind it

A vault is not a filing cabinet you organize for its own sake. It is a **second brain**. Its value is not tidy folders, it is connections. When your English essay links to your class notes which link to your book note, you have built a little web of your own understanding. Six months later that web is still there, and it is yours, in plain text, not rented from an app that might change or disappear.

## Do not do these (common ways people quit)

- **Do not organize for hours.** Organizing is not studying and it is not learning. Capture first, tidy later, tidy lightly.
- **Do not make it perfect.** A messy note that exists beats a perfect note you never wrote. Nobody grades your folders.
- **Do not try to use every folder on day one.** Start with the daily note and Classes. Add the rest when you actually need it.

## When something feels like too much

Drop back to the one habit: open the daily note, write two sentences. Momentum beats perfection every single time.
