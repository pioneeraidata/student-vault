# ⚙️ Setup: Turn On These Plugins

This vault ships pre-configured, so most of this is already done. This page is here in case something is off, or you are setting it up on a new computer. Takes three minutes.

Everything here uses **core plugins** that ship inside Obsidian. No downloads, no accounts, nothing that can break. That is on purpose, so this vault opens on any machine and just works.

## Already set up for you

- **Daily Notes** points to the `01 - Daily` folder and uses the Daily Note template automatically.
- **Templates** points to the `90 - System/Templates` folder.

## Do this once: two hotkeys

Hotkeys are what make the daily habit frictionless. Go to Settings (gear icon) then Hotkeys, search for these, and set a key you will remember:

1. **"Daily notes: Open today's daily note"**: try `Cmd/Ctrl + D`. This is the one you will use every day.
2. **"Templates: Insert template"**: try `Cmd/Ctrl + T`. This drops any template into the note you are in.

## If Daily Notes or Templates are not working

1. Settings then Core plugins. Make sure **Daily notes** and **Templates** are both on.
2. Settings then Daily notes. Set:
   - New file location: `01 - Daily`
   - Template file location: `90 - System/Templates/Daily Note`
3. Settings then Templates. Set Template folder location to `90 - System/Templates`.

## Optional, only if you want it later

- **Sync across devices:** Obsidian Sync is the official paid option and it is worth it if you want this vault on your phone and laptop in sync. A free alternative is putting the vault folder in iCloud, Google Drive, or Dropbox.
- **Auto-building dashboards:** a community plugin called Dataview can make the Assignments table build itself. It is powerful but optional. The by-hand tables in this vault work fine forever and never break. Add complexity only when you miss it.
